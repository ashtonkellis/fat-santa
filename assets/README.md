# Assets

Card-design assets downloaded from **Dextrous** (the board/card game design
tool this repo's `card-generator.html` is modelled on).

## Source

- Publisher: **Dextrous** — <https://www.dextrous.com.au/>
- itch.io page: <https://dextrousgamedesign.itch.io/>
- Both packs were offered as **free ("name your own price")** downloads and were
  downloaded at $0.

## Contents

### `card-frames/` — "Card Game Templates"
Source: <https://dextrousgamedesign.itch.io/card-game-templates>

12 blank decorative card frames, poker size (2.5" × 3.5"), PNG at 750 × 1050 px
(300 DPI). Filenames: `Bidding`, `DnD`, `Fresh mint`, `Geometric`,
`Into the blue`, `Intrigue`, `Magic wood`, `Quasar`, `Sci-fi salt`,
`Simple pixel`, `Simple`, `Templar`. `card game templates.jpg` is the pack's
preview/contact sheet.

### `blank-playing-cards/` — "Blank Playing Cards"
Source: <https://dextrousgamedesign.itch.io/blank-playing-cards>

6 print-ready blank playing-card sheets as PDFs, in three styles
(classic / dark / modern) each in A4 and US Letter. `playing cards.jpg` is the
preview.

## Licensing note

The itch.io pages did **not** state an explicit license. The packs are tagged
"No generative AI was used" and are described as free templates for card making
and prototyping. No formal grant of **redistribution** rights is published, so
before making this repository public or shipping these assets in a product,
confirm the usage terms with the creator (Dextrous). They are included here for
prototyping use with the card generator in this repo.
